Cairn Extension

Routes your local Cairn memory into web AI chats. Talks to a Cairn instance on your machine over HTTP.

Cairn endpoint

Default port 7716 is always-on. If your Cairn is older it might still be on 7717; the extension auto-falls-back. The auth token is only needed if you launched Cairn with CAIRN_API_TOKEN=….

Recall

Even with passive mode on, typing the prefix at the start of the composer opens the picker immediately. @cairn 投资人

Enabled sites

Toggle which chat sites get the Cairn UI. Disabled sites keep the right-click menu but skip composer / save-reply injection.
